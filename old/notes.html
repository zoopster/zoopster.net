Setup Uyuni

  • Install openSUSE 42.3
  • Add the repo - zypper ar -f http://download.opensuse.org/repositories/systemsmanagement:/Uyuni:/Master/openSUSE_Leap_42.3/systemsmanagement:Uyuni:Master.repo
  • Add the repo - zypper ar -f http://download.opensuse.org/repositories/systemsmanagement:/Uyuni:/Master:/Other/openSUSE_Leap_42.3/systemsmanagement:Uyuni:Master:Other.repo
  • Add the pattern - zypper in patterns-uyuni_server
  • Updates? Use zypper dup
